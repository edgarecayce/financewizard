# financewizard

## Purpose

`financewizard` is a project repo for personal investing tools, watchlists, alerts, and financial decision support workflows.

Use it for durable project context, requirements, decisions, notes, and implementation work related to portfolio tracking and investment analysis.

## Workflow

- Keep docs concise and current.
- Record important decisions and assumptions.
- Use direct GitHub actions for lightweight docs, notes, and decisions.
- Use branch/PR or Cursor/Claude for risky, destructive, multi-file, or code-heavy work.
