# Repo Brain Template

## Purpose
This repository is a reusable template for managing projects using a "repo as brain" approach.

## Structure
- docs/ → project context and status
- assets/ → images and reference material
- archive/ → old or unused content

## Workflow
- Update docs/current-status.md after each work session
- Track next actions in docs/next-steps.md
- Record important decisions in docs/decisions.md
